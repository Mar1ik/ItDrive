# ItDrive Frontend

Фронтенд для системы шеринговых поездок ItDrive.

## Технологии

- React 18.2.0
- React Router 6.20.0
- Axios 1.6.2
- React Query 3.39.3
- Vite 5.0.8

## Установка

```bash
cd frontend
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Сборка для production

```bash
npm run build
```

## Структура проекта

```
frontend/
├── src/

│   ├── App.jsx          # Главный компонент

```

## Функциональность

### Для всех пользователей
- Регистрация и авторизация
- JWT аутентификация
- Защищенные маршруты

### Для водителей
- Создание поездок
- Просмотр своих поездок
- Управление поездками (начало, завершение, отмена)
- Просмотр бронирований

### Для пассажиров
- Поиск доступных поездок
- Фильтрация по маршруту
- Бронирование мест
- Просмотр своих бронирований
- Отмена бронирований

### Для администраторов
- Просмотр статистики системы
- Просмотр популярных маршрутов
- Управление пользователями

## API

Фронтенд работает с REST API, запущенным на `http://localhost:8080/api`

Все запросы автоматически добавляют JWT токен в заголовок `Authorization`.

## Настройка

### API URL

Измените базовый URL API в `src/services/api.js` если необходимо:

```javascript
export const api = axios.create({
  baseURL: '/api', // или полный URL к вашему API
  ...
})
```

### 2GIS Maps API

Для работы карт необходимо настроить 2GIS Maps API:

1. Перейдите на [2GIS Developers Portal](https://dev.2gis.com/)
2. Войдите в свой аккаунт или зарегистрируйтесь
3. Создайте новый проект или выберите существующий
4. Получите API ключ в разделе "API ключи"
5. Замените `YOUR_API_KEY` в `index.html` на ваш API ключ:

```html
<script src="https://map.2gis.com/api?key=ВАШ_API_КЛЮЧ"></script>
```

**Примечание:** Если API ключ не настроен, карты не будут загружаться, но остальной функционал будет работать.

Подробная инструкция: см. `2GIS_MAPS_SETUP.md`
