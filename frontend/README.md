# ItDrive Frontend

Фронтенд для системы шеринговых поездок ItDrive.

## Технологии

- React 18.2.0
- React Router 6.20.0
- Axios 1.6.2
- React Query 3.39.3
- Vite 5.0.8

## Установка

```bash
cd frontend
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Сборка для production

```bash
npm run build
```

## Функциональность

### Для всех пользователей
- Регистрация и авторизация
- JWT аутентификация
- Защищенные маршруты

### Для водителей
- Создание поездок
- Просмотр своих поездок
- Управление поездками (начало, завершение, отмена)
- Просмотр бронирований

### Для пассажиров
- Поиск доступных поездок
- Фильтрация по маршруту
- Бронирование мест
- Просмотр своих бронирований
- Отмена бронирований

### Для администраторов
- Просмотр статистики системы
- Просмотр популярных маршрутов
- Управление пользователями

## API

Фронтенд работает с REST API, запущенным на `http://localhost:8080/api`

Все запросы автоматически добавляют JWT токен в заголовок `Authorization`.

## Настройка

### API URL

Измените базовый URL API в `src/services/api.js` если необходимо:

```javascript
export const api = axios.create({
  baseURL: '/api', // или полный URL к вашему API
  ...
})
```

### Google Maps API

Для работы карт необходимо настроить Google Maps API:

1. Перейдите на [Google Cloud Console](https://console.cloud.google.com/)
2. Создайте новый проект или выберите существующий
3. Включите следующие API:
   - Maps JavaScript API
   - Directions API
   - Places API
4. Создайте API ключ в разделе "Credentials"
5. Ограничьте ключ по HTTP referrers для безопасности (например, `localhost:3000/*`)
6. Замените `YOUR_API_KEY` в `index.html` на ваш API ключ:

```html
<script
  src="https://maps.googleapis.com/maps/api/js?key=ВАШ_API_КЛЮЧ&libraries=places,directions"
  async
  defer
></script>
```

**Примечание:** Если API ключ не настроен, карты не будут загружаться, но остальной функционал будет работать.
