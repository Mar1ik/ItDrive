# ItDrive
<br>EN</br> 
An application that allows users, namely students and staff of ITMO University, to travel to the university with the greatest benefit for both parties (carrier and passenger). 

<br>RU</br> 
Приложение которое позволяет пользователям, а именно студентам и сотрудникам университета ИТМО, перемещаться к университету с наибольшей выгодой для обеих сторон (перевозчика и пассажира).
